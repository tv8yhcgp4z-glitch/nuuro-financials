# Nuro — Your Money's Brain

AI-powered freelance finance app. Track income, estimate taxes with real 2024 brackets, manage clients, set goals, and get AI advice that actually knows your numbers.

## Features
- AI Finance Assistant — powered by Claude
- Smart Tax Engine — real federal brackets, all 50 states, SE tax, quarterly tracker
- Client Tracker — auto-groups income by client
- Goals & Deadlines — set targets with due dates
- Smart Import — upload .xlsx/.csv, auto-categorize
- Invoice Generator, Recurring Transactions, Deduction Finder, Tax Summary PDF

## Deploy to Vercel
1. Fork or clone this repo
2. Connect to Vercel
3. Add environment variable: ANTHROPIC_API_KEY = your API key
4. Deploy
